## Usage

You'll probably want to have two terminals open in your project's directory.

1. Start the auto-compiler.

```bash
$ lein hlisp auto
```

2. Start a local HTTP server.

```bash
$ cd resources/public
$ python -m SimpleHTTPServer
```

3. View the application at http://localhost:8000.
